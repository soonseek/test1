# 코드 사양 (Code Specification)

**Task ID**: task-1-1-2
**제목**: PokeAPI 기본 URL 및 엔드포인트 상수 관리 파일 생성

---

### 파일: `src/lib/constants.ts`
---
설명: PokeAPI의 기본 URL과 주요 엔드포인트 경로를 상수로 관리하는 파일입니다. API 주소 변경이나 버전 업데이트 시 이 파일만 수정하면 되도록 하여 유지보수성을 높입니다.
- **`POKEAPI_BASE_URL`**: PokeAPI v2의 기본 URL(`https://pokeapi.co/api/v2/`)을 문자열로 정의합니다.
- **`POKEAPI_ENDPOINTS`**: API의 하위 경로(엔드포인트)를 객체 형태로 정의합니다.
  - `pokemon`: 포켓몬 기본 정보 경로 (`pokemon`)
  - `pokemonList`: 포켓몬 목록 조회 경로 (`pokemon?limit={limit}&offset={offset}` 형태를 가정하므로 기본 경로만 정의)
- **TypeScript 타입 안전성**: 상수 값에 대한 타입이 명시적으로 추론되도록 `as const`를 사용하여 불변성 보장 (선택 사항이나 권장).

# 코드 사양 (Code Specification)

**Task ID**: task-1-1-1
**주제**: 포켓몬 데이터 타입 정의 (TypeScript Interfaces)

---

### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI 데이터와 클라이언트 상태를 위한 TypeScript 인터페이스를 정의하는 타입 정의 파일입니다. API 응답 형식을 표준화하고 애플리케이션 전역에서 재사용합니다.

- **`PokemonListItem` 인터페이스 정의**:
  - `id`: number (포켓몬 고유 ID)
  - `name`: string (포켓몬 이름)
  - `imageUrl`: string (포켓몬 이미지 URL)
- **`PokemonList` 인터페이스 정의**:
  - `results`: `PokemonListItem[]` (포켓몬 목록 배열)
  - `count`: number (전체 포켓몬 개수)
  - `next`: string | null (다음 페이지 URL 또는 null)
- **`PokemonType` 인터페이스 정의** (확장성 고려):
  - `name`: string (타입 이름, 예: fire, water)
  - `url`: string (상세 정보 URL)
- **`PokemonTypes` 인터페이스 정의** (확장성 고려):
  - `slot`: number
  - `type`: `PokemonType`

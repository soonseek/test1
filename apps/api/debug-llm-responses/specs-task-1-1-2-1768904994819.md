# 코드 사양 (Code Specification)

**Task ID**: task-1-1-2
**제목**: PokeAPI URL에서 ID 추출 로직 구현

---

### 파일: `src/lib/utils/extractIdFromUrl.ts`
---
설명: PokeAPI의 `url` 필드(예: `https://pokeapi.co/api/v2/pokemon/1/`)에서 마지막 슬래시(`/`)를 제거하고 순수 ID(숫자)를 추출하는 유틸리티 함수를 정의합니다.
- `extractIdFromUrl(url: string): number` 함수를 export 합니다.
- URL 구조 분석: `https://pokeapi.co/api/v2/pokemon/{id}/` 형식을 가정합니다.
- 문자열 조작 로직:
  1. 문자열 우측의 슬래시(`/`)를 제거합니다. (예: `.../1/` -> `.../1`)
  2. 문자열을 `/` 구분자로 분할(Split)합니다.
  3. 분할된 배열의 마지막 요소(ID)를 반환합니다.
- 반환 타입은 `number`입니다.
- 유효하지 않은 URL 형식이 들어올 경우를 대비해 기본값(예: `0`)을 반환하거나 에러 처리를 고려합니다.

### 파일: `src/lib/utils/extractIdFromUrl.test.ts`
---
설명: `extractIdFromUrl` 함수의 동작을 검증하는 단위 테스트 파일입니다.
- Jest 또는 Vitest 환경을 가정합니다.
- **Case 1**: 정상적인 URL (`https://pokeapi.co/api/v2/pokemon/25/`) 입력 시 `25` 반환 확인.
- **Case 2**: 끝 슬래시가 없는 URL (`https://pokeapi.co/api/v2/pokemon/25`) 입력 시 `25` 반환 확인.
- **Case 3**: 빈 문자열 또는 잘못된 형식 입력 시 안전한 값 반환 확인.

# 코드 사양 (Code Specification)

**Task ID**: task-1-2-2
**제목**: PokemonCard 이미지 영역 구현 (Next.js Image)

---

### 파일: src/components/pokemon/PokemonCard.tsx
---
설명: 포켓몬의 정보(이미지, 번호, 이름)를 시각적으로 표현하는 카드 컴포넌트입니다. 클릭 가능한 링크 역할을 수행하며, 반응형 그리드 레이아웃 내에서 렌더링됩니다.
- `src/components/ui/card` (shadcn/ui)의 `Card`, `CardHeader`, `CardContent` 컴포넌트를 조합하여 구조를 형성합니다.
- `src/components/pokemon/PokemonImage.tsx` 컴포넌트를 임포트하여 이미지 영역을 렌더링합니다.
- props로 `id`, `name`, `imageUrl`을 받습니다.
- 카드 전체를 클릭 시 `/pokemon/${id}` 경로로 이동하는 `Link` 컴포넌트로 감싸거나 내부에 배치합니다.
- Tailwind CSS를 사용하여 호버(Hover) 시 트랜지션 효과를 적용합니다.

### 파일: src/components/pokemon/PokemonImage.tsx
---
설명: Task 1-2-2의 핵심 컴포넌트로, Next.js의 `Image` 최적화 기능을 사용하여 포켓몬 이미지를 렌더링합니다.
- `next/image`에서 `Image` 컴포넌트를 임포트합니다.
- props로 `imageUrl` (소스 경로), `name` (대체 텍스트용), `width`, `height`를 받습니다.
- `src` 속성에 `imageUrl`을 적용합니다.
- `alt` 속성에 접근성(Accessibility)을 위해 포켓몬 `name`을 설정합니다.
- 이미지 로딩 최적화를 위해 `width`와 `height`를 명시적으로 설정합니다 (예: 200x200 또는 300x300).
- 이미지가 로드되기 전이거나 실패했을 때의 UI를 개선하기 위해 `placeholder="blur"` 또는 스타일링된 배경을 적용합니다.
- Tailwind CSS 클래스를 사용하여 이미지가 컨테이너 내에서 중앙 정렬되도록 설정합니다.

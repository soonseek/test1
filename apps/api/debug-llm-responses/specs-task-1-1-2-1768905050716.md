# 코드 사양 (Code Specification)

**Task ID**: task-1-1-2
**제목**: PokeAPI URL에서 ID 추출 로직 구현

---

### 파일: `src/lib/utils/extractIdFromUrl.ts`
---
설명: PokeAPI의 리스트 응답에 포함된 URL 문자열에서 포켓몬의 고유 ID를 숫자(Number) 타입으로 변환하여 반환하는 유틸리티 함수입니다.

- **함수명**: `extractIdFromUrl`
- **입력 파라미터**: `url: string` (예: `https://pokeapi.co/api/v2/pokemon/1/`)
- **반환값**: `number` (예: `1`)
- **구현 로직**:
  1. 문자열 우측의 슬래시(`/`)를 제거합니다.
  2. 문자열을 `/` 구분자로 분할(Split)하여 배열을 생성합니다.
  3. 분할된 배열의 마지막 요소(ID 문자열)를 추출합니다.
  4. 추출된 문자열을 `Number()`로 변환하여 반환합니다.
- **에러 처리**: 유효하지 않은 URL 형식이거나 ID가 숫자로 변환되지 않는 경우 `NaN`을 반환하거나, 로직에 따라 `0` 또는 예외를 처리할 수 있습니다 (본 태스크에서는 기본 변환 로직에 집중).
- **용도**: 포켓몬 리스트 데이터를 가져온 후, 상세 페이지 이동 경로(`href`)나 이미지 URL 생성을 위해 ID가 필요할 때 사용됩니다.

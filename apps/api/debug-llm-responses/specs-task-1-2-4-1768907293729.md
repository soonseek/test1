# 코드 사양 (Code Specification)

**Task ID**: task-1-2-4
**제목**: PokemonCard 반응형 스타일 적용 (Tailwind)

---

### 파일: `src/components/pokemon-card.tsx`
---
설명: 포켓몬의 정보(이미지, 번호, 이름)를 표시하는 개별 카드 컴포넌트이며, 반응형 및 인터랙티브 스타일링이 적용되는 핵심 UI 요소입니다.
- **반응형 레이아웃**: Tailwind CSS의 유틸리티 클래스를 사용하여 부모 컨테이너의 크기에 따라 유연하게 대응하도록 수정 (예: `w-full`, `max-w-xs`).
- **인터랙션 효과 (Hover)**:
  - `hover:shadow-xl` 또는 `hover:shadow-2xl` 클래스를 사용하여 마우스 오버 시 그림자를 진하게 표시.
  - `hover:-translate-y-1` 또는 `hover:-translate-y-2` 클래스를 사용하여 카드가 살짝 떠오르는 효과(상승) 적용.
  - `transition-all duration-300` 등을 추가하여 부드러운 애니메이션 전환 구현.
- **스타일링**:
  - `rounded-xl` 또는 `rounded-2xl`을 사용하여 둥근 모서리 디자인 유지.
  - `bg-white` (또는 `bg-card`) 배경색과 테두리(`border`) 정의.
  - 내부 컨텐츠(이미지, 텍스트)의 정렬 및 간격(`gap`, `padding`) 조정.
- **링크 연결**: 카드 전체 또는 특정 영역을 클릭 시 상세 페이지(`/pokemon/[id]`)로 이동하도록 `<Link>` 컴포넌트로 감싸거나 `onClick` 이벤트 처리.

### 파일: `src/app/page.tsx`
---
설명: 포켓몬 카드 리스트를 그리드 형태로 배치하는 메인 페이지로, 카드 컴포넌트의 반응형 동작을 위한 그리드 시스템을 정의합니다.
- **그리드 레이아웃**:
  - `grid grid-cols-1` (모바일 기본)
  - `sm:grid-cols-2` (소형 화면)
  - `md:grid-cols-3` (중형 화면)
  - `lg:grid-cols-4` (대형 화면)
  - `xl:grid-cols-5` (초대형 화면) 등을 사용하여 다양한 뷰포트 지원.
- **간격 설정**: `gap-4` 또는 `gap-6`을 사용하여 카드 간의 적절한 여백 확보.
- **데이터 렌더링**: `PokemonCard` 컴포넌트를 매핑하여 렌더링하며, 각 카드에 고유한 `key` (포켓몬 ID) 전달.

### 파일: `src/components/ui/card.tsx` (shadcn/ui)
---
설명: 프로젝트에 설치된 shadcn/ui Card 컴포넌트의 기본 스타일을 확인하거나 확장합니다. (이미 존재한다고 가정하며, 커스텀이 필요할 경우 수정 대상)
- `PokemonCard` 내부에서 shadcn/ui의 `Card`, `CardHeader`, `CardContent` 등을 사용할 경우, 해당 컴포넌트들이 Tailwind의 `className` prop을 통해 커스터마이징 가능한지 확인.
- 필요시 `src/components/ui/` 내부의 스타일 변수(CSS Variables)가 프로젝트 테마(노란색, 파란색)와 일치하도록 설정 확인.

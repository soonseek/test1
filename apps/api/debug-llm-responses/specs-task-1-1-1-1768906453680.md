# 코드 사양 (Code Specification)

**Task ID**: task-1-1-1
**제목**: PokeAPI 응답 데이터 타입 정의

---

### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI v2의 응답 구조와 애플리케이션에서 사용하는 포켓몬 도감 데이터의 형식을 정의하는 TypeScript 인터페이스 파일입니다.

- **`IPokemonListResponse` 인터페이스 정의**
  - PokeAPI `https://pokeapi.co/api/v2/pokemon?limit=...` 호출 시 반환되는 JSON 구조와 매핑합니다.
  - `count`: 전체 포켓몬 개수 (number)
  - `next`: 다음 페이지 URL (string | null)
  - `previous`: 이전 페이지 URL (string | null)
  - `results`: 포켓몬 요소 배열 (`IPokemonListItem[]`)

- **`IPokemonListItem` 인터페이스 정의**
  - 리스트 응답 내부의 개별 항목 구조를 정의합니다.
  - `name`: 포켓몬 이름 (string)
  - `url`: 상세 정보 조회 URL (string)

- **`IPokemon` 인터페이스 정의**
  - 앱 UI(카드, 상세 페이지)에서 사용할 도메인 모델 형식입니다.
  - `id`: 포켓몬 고유 번호 (number)
  - `name`: 포켓몬 이름 (string)
  - `imageUrl`: 포켓몬 이미지 주소 (string, `raw.githubusercontent.com` 등의 외부 리소스 또는 로컬 경로)
  - (선택) `types`: 포켓몬 타입 정보 (향후 확장을 고려한 구조)

- **타입 가드 및 유틸리티 타입 (선택 사항)**
  - API 응답 데이터를 `IPokemon` 모델로 변환하는 로직(Mapper)을 위한 입력 타입으로 활용될 수 있도록 명확한 필드 명시

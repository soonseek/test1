# 코드 사양 (Code Specification)

**Task ID**: task-1-2-4
**제목**: 포켓몬 ID 포맷팅 유틸 함수 구현

---

### 파일: src/lib/utils.ts
---
설명: 프로젝트 전반에서 사용되는 범용 유틸리티 함수들을 모아두는 파일입니다. Tailwind CSS의 `cn` 함수와 함께, 포켓몬의 숫자 ID를 UI 표시용 포맷(예: #001)으로 변환하는 함수를 추가합니다.

- **`formatPokemonId(id: number | string): string` 함수 구현**
  - 입력받은 `id`를 숫자로 처리합니다.
  - `String.prototype.padStart(3, '0')` 메서드를 사용하여 3자리 문자열로 변환합니다 (예: `1` → `"001"`, `25` → `"025"`).
  - 변환된 문자열 앞에 `#` 기호를 붙여 반환합니다 (예: `#001`).
  - `id`가 `undefined`이거나 유효하지 않은 경우 안전하게 처리하거나 빈 문자열을 반환합니다.
- 기존 `cn` (classnames utility) 함수는 그대로 유지합니다.

### 파일: src/lib/__tests__/utils.test.ts (선택 사항)
---
설명: 유틸리티 함수의 동작을 검증하기 위한 테스트 파일입니다 (Jest 또는 Vitest 사용 가정).

- `formatPokemonId` 함수의 다양한 입력 경우에 대한 테스트 케이스 작성:
  - 1자리 숫자 (1 → "#001")
  - 2자리 숫자 (25 → "#025")
  - 3자리 숫자 (150 → "#150")
  - 0 또는 잘못된 입력 처리

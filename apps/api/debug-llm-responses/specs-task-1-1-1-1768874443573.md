### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI v2의 응답 데이터 구조에 맞춰 타입 안전성을 보장하기 위한 TypeScript 인터페이스 정의 파일입니다. 이 파일은 추후 데이터 페칭 로직(`fetchPokemonList`) 및 컴포넌트 렌더링 시 사용될 예정입니다.

- **`PokemonListItem` 인터페이스 정의**:
  - `name` (string): 포켓몬의 이름 (예: "bulbasaur")
  - `url` (string): 해당 포켓몬의 상세 정보 API 주소 (예: "https://pokeapi.co/api/v2/pokemon/1/")
  - (선택 사항) URL에서 ID를 추출하기 위한 유틸리티 타입 또는 로직 고려 가능

- **`PokemonListResponse` 인터페이스 정의**:
  - `count` (number): 전체 포켓몬 개수
  - `next` (string | null): 다음 페이지 URL
  - `previous` (string | null): 이전 페이지 URL
  - `results` (`PokemonListItem[]`): 포켓몬 목록 데이터 배열

- **내보내기(Export)**:
  - 다른 모듈(`src/lib/api/pokemon.ts` 등)에서 사용할 수 있도록 `export` 키워드 사용

# 코드 사양 (Code Specification)

**Task ID**: task-1-1-1
**제목**: PokeAPI 응답 타입 인터페이스 정의

---

### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI v2의 응답 구조에 맞춰 포켓몬 데이터의 타입을 정의하는 파일입니다. 이 타입들은 API 호출 함수 및 컴포넌트에서 재사용됩니다.

- **`PokemonListItem` 인터페이스 정의**:
  - `name` (string): 포켓몬의 이름 (예: "bulbasaur")
  - `url` (string): 해당 포켓몬의 상세 정보 API URL (예: "https://pokeapi.co/api/v2/pokemon/1/")
  - (선택 사항) URL에서 ID를 추출하기 위한 유틸리티 타입 또는 로직을 고려한 구조

- **`PokemonListResponse` 인터페이스 정의**:
  - `count` (number): 전체 포켓몬 개수
  - `next` (string | null): 다음 페이지 URL
  - `previous` (string | null): 이전 페이지 URL
  - `results` (PokemonListItem[]): 포켓몬 목록 배열

- **내보내기 (Export)**:
  - 정의된 인터페이스들을 `export` 키워드를 사용하여 다른 모듈(Server Actions, Components)에서 import 할 수 있도록 합니다.

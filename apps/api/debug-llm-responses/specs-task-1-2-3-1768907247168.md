### 파일: `src/components/pokemon/PokemonCard.tsx`
---
설명: 포켓몬 카드의 단일 UI를 담당하는 컴포넌트입니다. 이미지와 함께 하단에 위치할 텍스트 정보 영역(번호, 이름)을 구현합니다.
- `src/components/ui/card` (shadcn/ui)를 기반으로 하거나, `div`와 `Tailwind CSS`를 사용하여 카드 형태를 구성합니다.
- **번호(ID) 표시**: `text-sm`, `text-gray-500` (또는 `text-muted-foreground`) 스타일을 적용하여 작고 회색 텍스트로 표시합니다. (예: `#001`)
- **이름(Name) 표시**: `text-xl`, `font-bold` 스타일을 적용하여 크고 굵은 텍스트로 표시합니다.
- 레이아웃은 이미지 하단에 위치하며, `flex-col` 또는 `block` 방식으로 정보를 수직 배치합니다.
- 향후 클릭 이벤트 및 링크 연결을 대비하여 `group` 클래스나 `cursor-pointer` 속성을 고려하여 구조를 잡습니다.

### 파일: `src/components/pokemon/PokemonImage.tsx`
---
설명: (참고) 카드 내에서 사용될 포켓몬 이미지 전용 컴포넌트입니다. 텍스트 영역과의 시각적 균형을 맞춥니다.
- Next.js의 `Image` 컴포넌트를 사용하여 포켓몬 이미지를 최적화하여 렌더링합니다.
- `width`, `height` prop을 받아 크기를 조절하며, 기본값은 카드 크기에 맞게 설정합니다.
- 이미지 로딩 중이거나 실패했을 때를 대비한 fallback UI(placeholder)를 포함합니다.

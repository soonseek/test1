# 코드 사양 (Code Specification)

**Task ID**: task-1-1-1
**제목**: PokeAPI 응답 데이터 타입 정의

---

### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI v2의 응답 구조와 매핑되는 TypeScript 인터페이스를 정의합니다. 이 타입들은 API Route 및 클라이언트 컴포넌트 전체에서 재사용됩니다.
- **PokeAPIListResponse**: PokeAPI `https://pokeapi.co/api/v2/pokemon?limit=...` 엔드포인트의 원본 응답 형식을 정의합니다. (`count`, `next`, `previous`, `results` 포함)
- **PokeAPIListItem**: 리스트 내 개별 아이템의 형식을 정의합니다. (`name`, `url` 포함)
- **PokemonListItem**: 클라이언트/UI 단에서 사용하기 위해 가공된 형태를 정의합니다. (`id`, `name`, `image` 등을 포함하도록 설계하여 향후 확장성 확보)

### 파일: `src/lib/utils/pokemon.ts`
---
설명: PokeAPI 데이터를 다루는 유틸리티 함수를 모아둔 모듈입니다. (향후 Task를 위한 준비)
- **extractIdFromUrl**: PokeAPI 리스트 아이템의 `url` 속성(예: `https://pokeapi.co/api/v2/pokemon/1/`)을 파싱하여 숫자형 ID(예: `1`)를 추출하는 함수를 정의합니다.
- 정규식(Regex) 또는 URL 생성자를 사용하여 안전하게 ID를 반환하도록 구현합니다.

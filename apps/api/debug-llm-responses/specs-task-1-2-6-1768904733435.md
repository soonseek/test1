# 코드 사양 (Code Specification)

**Task ID**: task-1-2-6
**제목**: 카드 호버(Hover) 효과 및 스타일링 적용
**설명**: `PokemonCard` 컴포넌트에 Tailwind CSS 클래스를 적용하여 인터랙티브한 호버 효과(확대, 그림자, 전환 애니메이션)를 구현합니다.

---

### 파일: `src/components/pokemon-card.tsx`
---
설명: 포켓몬의 정보(이미지, 이름, 번호)를 표시하는 카드 컴포넌트로, 사용자 인터랙션에 반응하는 스타일링을 적용합니다.
- **Tailwind CSS 클래스 적용**:
  - `group` 또는 컨테이너 요소에 `transition-transform`, `duration-300`, `ease-in-out` 클래스를 추가하여 부드러운 움직임 구현
  - 호버 상태(`hover:`)에서 `scale-105` 클래스를 적용하여 카드가 5% 확대되는 효과 적용
  - 호버 상태(`hover:`)에서 `shadow-xl` 또는 `shadow-2xl` 클래스를 적용하여 카드 위로 떠오르는 듯한 그림자 효과 추가
- **접근성 고려**:
  - `focus-visible:` 상태에서도 호버 효과와 유사한 시각적 피드백(스케일 또는 아웃라인)을 제공하여 키보드 네비게이션 지원
- **구조**:
  - `Link` 컴포넌트를 최상위 래퍼로 사용하여 클릭 시 상세 페이지로 이동
  - 내부에 `Image`(포켓몬 이미지), `Text`(이름, ID) 배치

### 파일: `src/app/page.tsx`
---
설명: 포켓몬 목록을 그리드 형태로 렌더링하는 메인 페이지로, 스타일링이 적용된 카드 컴포넌트를 사용합니다.
- `PokemonCard` 컴포넌트를 import 하여 사용
- `grid` 레이아웃(`grid-cols-2`, `md:grid-cols-4` 등) 내부에 카드 배치
- 각 카드에 고유한 `key` prop 전달
- (참고) 이 파일의 로직은 변경하지 않으나, 카드가 정상적으로 렌더링되는지 확인하는 용도로 사양에 포함

### 파일: `tailwind.config.ts`
---
설명: 프로젝트 전체의 Tailwind CSS 설정을 관리하는 파일입니다.
- `content` 배열에 `src/components/pokemon-card.tsx` 경로가 포함되어 있는지 확인 (JIT 모드 동작 보장)
- 프로젝트 테마(노란색, 파란색)와 어울리는 그림자 색상이 필요할 경우 `extend` 내의 `colors` 또는 `boxShadow`를 확장하여 사용할 수 있음 (선택 사항)

### 파일: `src/components/pokemon-card.tsx`
---
설명: 포켓몬의 정보를 시각화하는 컴포넌트로, 사용자의 상호작용(호버, 클릭)에 반응하는 스타일을 적용합니다.
- **기존 기능 유지**: 포켓몬 이미지, 번호(ID), 이름 표시
- **인터랙션 스타일 추가**:
  - `hover:scale-105`: 마우스 오버 시 카드가 5% 확대되어 부풀어 오르는 효과 적용
  - `transition-transform`: 확대 효과가 부드럽게 진행되도록 트랜지션 설정
  - `cursor-pointer`: 마우스 커서를 손가락 모양으로 변경하여 클릭 가능함을 시각적으로 표시
- **접근성 고려**: 키보드 포커스 시에도 유사한 시각적 피드백 제공 (예: `focus:scale-105` 등)
- **링크 연동**: `Link` 컴포넌트로 감싸거나 클릭 이벤트 핸들러를 통해 상세 페이지(`/pokemon/[id]`)로 이동

### 파일: `src/app/page.tsx`
---
설명: 메인 페이지로, 스타일이 업데이트된 `PokemonCard` 컴포넌트를 그리드 형태로 렌더링합니다.
- `PokemonCard` 컴포넌트를 import하여 사용
- 반응형 그리드 레이아웃(`grid-cols-2`, `md:grid-cols-4` 등) 유지
- 각 카드에 고유한 `key` prop 전달
- 서버 컴포넌트(Server Component)로서 데이터 페칭 로직 유지

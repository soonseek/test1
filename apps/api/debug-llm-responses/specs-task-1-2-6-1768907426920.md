# 코드 사양 (Code Specification)

**Task ID**: task-1-2-6
**제목**: 그리드 레이아웃 CSS 구성 (Grid System)

---

### 파일: `src/app/page.tsx`
---
설명: 포켓몬 목록을 렌더링하는 메인 페이지 컨테이너
- Server Component로 동작하며 데이터 페칭 로직은 포함하지 않음 (UI 레이아웃 집중)
- `PokemonGrid` 컴포넌트를 렌더링하여 목록 표시
- 페이지 전체의 기본 패딩 및 최대 너비 설정 (`max-w-7xl mx-auto` 등)

### 파일: `src/components/pokemon-grid.tsx`
---
설명: 포켓몬 카드 목록을 반응형 그리드로 배치하는 컨테이너 컴포넌트
- Tailwind CSS Grid 유틸리티를 사용하여 레이아웃 구성
- **반응형 클래스 적용**:
  - 기본(Mobile): `grid-cols-2` (2열)
  - 중간(Tablet): `md:grid-cols-4` (4열)
  - 큰화면(Desktop): `lg:grid-cols-6` (6열)
- 아이템 간 간격(Gap) 설정: `gap-4`
- 자식 요소로 `PokemonCard` 컴포넌트를 반복 렌더링

### 파일: `src/components/pokemon-card.tsx`
---
설명: 개별 포켓몬 정보를 표시하는 카드 컴포넌트
- `shadcn/ui`의 `Card` 컴포넌트를 기반으로 구성 (또는 `div`와 Tailwind 클래스로 직접 스타일링)
- 카드 내부 콘텐츠 구조:
  - 이미지 영역: `Next.js Image` 컴포넌트 사용 (`fill` 또는 `responsive` 속성)
  - 정보 영역: 포켓몬 번호(ID), 이름 표시
- 호버(Hover) 효과 적용: `hover:scale-105 transition-transform` 등으로 상호작성 표시
- 클릭 시 상세 페이지(`Link`)로 이동

### 파일: `src/components/ui/card.tsx`
---
설명: shadcn/ui 기반의 카드 UI 기본 컴포넌트 (설치되어 있지 않은 경우 생성)
- `CardHeader`, `CardContent`, `CardFooter` 등의 하위 컴포넌트 포함
- 재사용 가능한 스타일링 제공
- `class-variance-authority` (CVA)를 사용하여 변형 관리

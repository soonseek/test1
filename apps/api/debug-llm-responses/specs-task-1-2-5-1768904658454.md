### 파일: src/types/pokemon.ts
---

  - `name`: string (포켓몬 이름)

### 파일: src/components/pokemon-card.tsx
---
설명: 포켓몬 정보를 시각화하는 개별 카드 컴포넌트 (Client Component)
- `src/types/pokemon.ts`에서 정의한 타입 또는 별도의 Props 인터페이스 정의
- Props 구성:
  - `id`: number (화면에 #001 형식으로 표시)
  - `name`: string (카드 하단에 텍스트로 표시)
  - `imageUrl`: string (이미지 소스로 사용)
- UI 구현:
  - `shadcn/ui`의 `Card` 컴포넌트 구조 사용 (`Card`, `CardHeader`, `CardContent` 등)
  - 이미지 영역: `next/image` 컴포넌트 사용
    - `src`: props.imageUrl
    - `alt`: props.name
    - `fill`: true (부모 요소에 맞춤)
    - `sizes`: 반응형 이미지 최적화 속성
    - `placeholder`: 'blur' (선택 사항, 로딩 경험 개선)
  - 텍스트 영역: 포켓몬 ID와 이름 표시
- 스타일링:
  - Tailwind CSS를 사용하여 호버(Hover) 시 그림자 효과 또는 스케일 애니메이션 추가
  - 반응형 대응을 위한 `w-full` 및 `aspect-square` 클래스 적용

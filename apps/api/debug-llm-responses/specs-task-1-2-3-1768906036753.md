# 코드 사양 (Code Specification)

**작성자**: Next.js 14+ Full-Stack Architect

### 파일: `src/app/page.tsx`
---
설명: 포켓몬 도감의 메인 페이지로, 서버 컴포넌트(Server Component)로 동작하며 데이터를 페칭하고 리스트 레이아웃을 구성합니다.
- `getPokemonList` 등의 데이터 페칭 로직(혹은 더미 데이터)을 호출하여 포켓몬 배열을 가져옵니다.
- `PokemonGrid` 컴포넌트에 데이터를 props로 전달하여 렌더링합니다.
- 페이지 전체의 메타데이터나 기본 레이아웃 구조를 정의합니다.

### 파일: `src/components/pokemon-grid.tsx`
---
설명: 포켓몬 카드 리스트를 감싸는 그리드 레이아웃 컨테이너입니다. Tailwind CSS의 Grid 시스템을 사용하여 반응형 디자인을 구현합니다.
- **Grid Layout**: `grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4` 클래스를 적용하여 화면 크기에 따라 열의 개수가 변하도록 설정합니다.
- **Gap**: `gap-4` 또는 `gap-6`을 설정하여 카드 간의 간격을 조정합니다.
- **Container**: `max-w-7xl mx-auto` 등을 사용하여 콘텐츠가 화면 중앙에 정렬되도록 합니다.
- `pokemonList` 데이터를 받아와 자식 컴포넌트인 `PokemonCard`를 매핑하여 표시합니다.

### 파일: `src/components/pokemon-card.tsx`
---
설명: 개별 포켓몬의 정보(이미지, 번호, 이름)를 표시하는 카드 컴포넌트입니다.
- **Link**: `next/link`를 사용하여 카드 클릭 시 상세 페이지(`/pokemon/[id]`)로 이동합니다.
- **Image**: `next/image` 컴포넌트를 사용하여 포켓몬 이미지를 최적화하여 표시합니다 (`fill` 또는 `width/height` prop 사용).
- **Content**: 포켓몬 번호(ID), 이름을 텍스트로 표시합니다.
- **Styling**: shadcn/ui의 `Card` 컴포넌트를 활용하거나 Tailwind 클래스를 사용하여 그림자, 호버 효과, 둥근 모서리 등을 적용합니다.

### 파일: `src/types/pokemon.ts`
---
설명: 포켓몬 데이터의 타입을 정의하는 TypeScript 인터페이스 또는 타입 정의 파일입니다.
- `Pokemon` 인터페이스를 정의합니다 (예: `id: number`, `name: string`, `imageUrl: string`, `types: string[]`).
- API 응답 타입과 내부 사용 타입을 명확히 구분하여 타입 안전성을 확보합니다.

### 파일: `tailwind.config.ts`
---
설명: Tailwind CSS의 설정 파일입니다 (이미 존재하지만, 프로젝트 테마에 맞춰 확인이 필요합니다).
- 프로젝트의 색상 테마(노란색, 파란색)가 `theme.extend`에 정의되어 있는지 확인합니다.
- 필요한 경우 shadcn/ui와의 호환성을 위한 baseColor 설정을 확인합니다.

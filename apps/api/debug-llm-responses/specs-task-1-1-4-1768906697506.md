# 코드 사양 (Code Specification)

**Task ID**: task-1-1-4
**제목**: 데이터 매핑 및 이미지 URL 변환 로직 추가

이 사양은 PokeAPI의 원시 데이터를 클라이언트 최적화 형태로 변환하는 로직을 정의합니다. ID 추출 및 공식 아트워크 이미지 URL 생성 로직을 포함합니다.

---

### 파일: src/types/pokemon.ts
---
설명: 포켓몬 데이터의 구조를 정의하는 TypeScript 인터페이스 모음
- **PokeApiListItem**: PokeAPI v2 `pokemon?limit=...` 응답의 단일 아이템 형태 (`name`, `url` 포함)
- **Pokemon**: 클라이언트에서 사용할 최종 데이터 형태 (`id`, `name`, `imageUrl` 포함)
- `id`는 숫자형, `imageUrl`은 문자열로 정의

### 파일: src/lib/utils/pokemonMapper.ts
---
설명: 데이터 변환 로직을 담당하는 유틸리티 함수 (순수 함수)
- **mapPokemonData(data: PokeApiListItem): Pokemon** 함수 export
- URL 파싱 로직: `https://pokeapi.co/api/v2/pokemon/{id}/` 형태의 URL에서 마지막 슬래시 뒤의 숫자(ID)를 추출
- 이미지 URL 생성 로직: 추출한 ID를 사용하여 `raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/{id}.png` 형식의 URL 생성
- 타입 안전성 보장 (입력 타입 검증)

### 파일: src/lib/__tests__/pokemonMapper.test.ts (선택 사항이지만 권장됨)
---
설명: 데이터 매핑 로직의 정확성을 검증하는 테스트 코드
- URL에서 ID가 정확히 추출되는지 테스트 (예: URL 끝이 `/1/`일 때 ID가 1인지)
- 이미지 URL이 포맷에 맞게 생성되는지 테스트
- 경계값 테스트 (ID가 0이거나 매우 클 경우)

# 코드 사양 (Code Specification)

작성일: 2026. 1. 20.
아키텍트: Next.js 14+ Full-Stack Architect

---

### 파일: src/app/api/pokemons/route.ts
---
설명: Next.js App Router의 Route Handler를 사용하여 포켓몬 목록 데이터를 제공하는 API 엔드포인트
- HTTP GET 요청을 처리하는 `GET` 함수를 비워둔 상태로 정의 (Skeleton)
- 향후 PokeAPI(v2)와의 통신 및 데이터 매핑 로직이 추가될 예정
- 응답 데이터 타입은 `Promise<Response>` 형태를 준수
- CORS 및 에러 핸들링을 위한 기본 구조预留

### 파일: src/types/pokemon.ts
---
설명: PokeAPI 응답 데이터 및 애플리케이션 내부에서 사용하는 포켓몬 데이터 형태를 정의하는 TypeScript 인터페이스
- `PokeAPIListResponse`: PokeAPI의 원본 리스트 응답 구조 (count, next, previous, results)
- `PokemonListItem`: 클라이언트에 반환할 간소화된 데이터 구조 (id, name, url)
- `PokemonType`: 포켓몬의 타입(예: fire, water)을 정의하는 문자열 유니언 타입
- 향후 상세 정보 타입 확장을 대비한 네이밍 규칙 적용

### 파일: src/lib/utils.ts
---
설명: 프로젝트 전반에서 사용되는 유틸리티 함수들의 모음 (shadcn/ui 설치 시 자동 생성됨)
- `cn` 함수: Tailwind CSS의 clsx 및 class-variance-authority를 활용한 클래스명 병합 유틸리티
- 향후 URL 파싱 및 ID 추출 로직이 추가될 수 있는 공간

### 파일: src/app/layout.tsx
---
설명: 애플리케이션의 루트 레이아웃을 정의하는 Server Component 파일
- shadcn/ui의 `Toaster` 컴포넌트를 포함하여 전역 UI 상태 관리 준비
- 폰트 및 메타데이터 설정
- 자식 컴포넌트(`children`)를 렌더링하는 기본 구조 유지

# 코드 사양 (Code Specification)

**Task ID**: task-1-1-1
**주제**: PokeAPI 응답 타입 인터페이스 정의

---

### 파일: `src/types/pokemon.ts`
---
설명: PokeAPI v2의 응답 구조에 맞춰 포켓몬 데이터의 타입 안전성을 보장하기 위한 TypeScript 인터페이스 정의 파일입니다. 이 파일은 추후 데이터 페칭 로직(`src/lib/api/pokemon.ts`) 및 UI 컴포넌트에서 참조됩니다.

- **`NamedAPIResource` 인터페이스 정의**:
  - PokeAPI의 공통 응답 형식인 `name`(string)과 `url`(string) 속성을 포함하는 베이스 인터페이스를 정의합니다.
  - 이 인터페이스는 향후 타입, 진화 체인 등 다른 API 리소스에서 재사용될 수 있습니다.

- **`PokemonListItem` 인터페이스 정의**:
  - `NamedAPIResource`를 상속받거나 포함하여 구성합니다.
  - 목록 조회 시 필요한 최소한의 정보(이름, URL)를 명시합니다.
  - (선택 사항이나 권장) URL에서 ID를 파싱하여 사용할 수 있도록 `id?: number` 속성을 추가로 정의하여 UI 표시 시 유용하게 사용합니다.

- **`PokemonListResponse` 인터페이스 정의**:
  - PokeAPI의 `GET /pokemon` 엔드포인트의 최상위 응답 구조를 정의합니다.
  - `count` (number): 전체 포켓몬 개수
  - `next` (string | null): 다음 페이지 URL
  - `previous` (string | null): 이전 페이지 URL
  - `results` (PokemonListItem[]): 포켓몬 목록 배열

- **내보내기(Export)**:
  - 위에서 정의한 모든 인터페이스를 `export` 키워드를 사용하여 외부 모듈에서 import 할 수 있도록 합니다.

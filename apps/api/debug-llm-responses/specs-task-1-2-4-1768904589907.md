# 코드 사양 (Code Specification)

**Task ID**: task-1-2-4
**제목**: 메인 리스트 레이아웃 그리드(Grid) 구성
**작성자**: Next.js 14+ Full-Stack Architect

---

## 개요
이 사양은 `STORY-1-2`의 인수 조건을 충족하기 위해 작성되었습니다. 포켓몬 카드 목록을 반응형으로 표시하기 위한 UI 컴포넌트 구조를 정의합니다. `shadcn/ui`의 `Card` 컴포넌트와 `Tailwind CSS`의 Grid 시스템을 활용하여 모바일부터 데스크톱까지 일관된 레이아웃을 제공합니다.

---

### 파일: src/components/pokemon/pokemon-grid.tsx
---
설명: 포켓몬 카드 목록을 반응형 그리드 레이아웃으로 렌더링하는 컨테이너 컴포넌트입니다.
- `props`로 포켓몬 데이터 배열(`Pokemon[]`)을 받습니다.
- **반응형 레이아웃 구현**:
  - 기본(Mobile): `grid-cols-1` (1열)
  - Small (sm): `sm:grid-cols-2` (2열)
  - Medium (md): `md:grid-cols-3` (3열)
  - Large (lg): `lg:grid-cols-4` (4열)
- **간격 설정**: `gap-4` 또는 `gap-6`을 적용하여 카드 간 여백을 조정합니다.
- `grid` 클래스와 `grid-flow-row`를 사용하여 자연스러운 흐름을 구성합니다.
- 데이터가 없을 경우 "표시할 포켓몬이 없습니다"와 같은 메시지를 처리합니다.

### 파일: src/components/pokemon/pokemon-card.tsx
---
설명: 개별 포켓몬 정보를 표시하는 카드 컴포넌트입니다 (shadcn/ui Card 기반).
- `shadcn/ui`의 `Card`, `CardHeader`, `CardContent` 등을 구성하여 사용합니다.
- **이미지 최적화**: `next/image`의 `Image` 컴포넌트를 사용하여 포켓몬 이미지를 렌더링합니다.
  - `fill` 속성 또는 명시적 `width/height` 사용
  - `placeholder="blur"`를 고려하여 로딩 시 깜빡임 방지
- **정보 표시**:
  - 포켓몬 번호(ID): `#001` 형식으로 포맷팅하여 표시 (예: `text-muted-foreground`)
  - 포켓몬 이름: 영문 이름을 표시하며, 대문자로 변환하거나 적절한 폰트 크기(`text-xl`, `font-bold`)로 강조
- **링크 연결**: 카드 전체 또는 특정 영역을 클릭 시 `/pokemon/[id]` 상세 페이지로 이동하도록 `Link` 컴포넌트로 감싸거나 `onClick` 이벤트를 처리합니다.

### 파일: src/app/page.tsx
---
설명: 메인 페이지로, 데이터를 페칭하고 `PokemonGrid` 컴포넌트에 데이터를 전달합니다.
- **Server Component**로 동작합니다.
- `getPokemonList` 등의 데이터 페칭 로직(API 또는 DB)을 호출하여 포켓몬 목록 데이터를 가져옵니다.
- 가져온 데이터를 `props`로 `PokemonGrid`에 전달합니다.
- 페이지 레벨에서의 기본 레이아웃(padding, max-width 등)을 정의하여 그리드가 화면 중앙에 위치하도록 합니다.

### 파일: src/types/pokemon.ts
---
설명: 포켓몬 데이터의 타입을 정의합니다 (타입 안정성 확보).
- `Pokemon` 인터페이스 또는 타입 정의:
  - `id`: number
  - `name`: string
  - `types`: string[] (향후 타입 표시를 위해 미리 정의)
  - `imageUrl`: string (이미지 URL)
- 이 타입은 `page.tsx`에서 데이터를 받을 때와 `pokemon-card.tsx`에서 props를 받을 때 사용됩니다.

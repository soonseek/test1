### 파일: `src/lib/utils.ts`
---
설명: 프로젝트 전반에서 사용되는 범용 유틸리티 함수들을 모아두는 파일입니다. Tailwind CSS 설정과 함께 포켓몬 ID 포맷팅 로직을 추가합니다.
- `cn` 함수 (clsx + tailwind-merge)가 이미 존재하는지 확인하고, 없을 경우 생성합니다.
- `formatPokemonId` 함수를 구현합니다:
  - **입력**: `id: number` (예: 1, 25, 151)
  - **동작**: 숫자를 3자리 문자열로 변환하고 앞에 `#` 기호를 붙입니다.
  - **구현 로직**: `String(id).padStart(3, '0')` 또는 유사한 로직 사용.
  - **출력**: `string` (예: "#001", "#025", "#151")
- 함수를 export 하여 컴포넌트에서 사용할 수 있도록 합니다.

# 코드 사양 (Code Specification)

**Task ID**: task-1-2-5
**대상 컴포넌트**: PokemonGrid (Container/Layout)

### 파일: `src/components/pokemon/pokemon-grid.tsx`
---
설명: 포켓몬 카드 목록을 반응형 그리드 레이아웃으로 배치하는 컨테이너 컴포넌트입니다. shadcn/ui의 디자인 시스템과 Tailwind CSS를 활용하여 구현합니다.

- **Props 정의**:
  - `children`: `ReactNode` 타입. 내부에 렌더링될 포켓몬 카드 컴포넌트들을 받습니다.
  - (선택 사항) `className`: 추가적인 Tailwind 클래스를 덧붙이기 위한 prop.
- **레이아웃 구현**:
  - Tailwind CSS Grid 시스템 사용 (`grid grid-cols-1 ...`).
  - **반응형 디자인**:
    - 모바일(sm): 1열 (`grid-cols-1`)
    - 태블릿(md): 2열 (`grid-cols-2`)
    - 데스크톱(lg): 3열 또는 4열 (`grid-cols-3` 또는 `grid-cols-4`)
  - 간격(Gap) 설정: `gap-4` 또는 `gap-6`을 사용하여 카드 간의 여백을 일정하게 유지합니다.
- **스타일링**:
  - 너비를 `w-full`로 설정하여 부모 컨테이너에 꽉 차게 배치합니다.
  - 필요시 `max-w-7xl mx-auto` 등을 통해 콘텐츠의 최대 너비를 제한하고 중앙 정렬합니다.
- **접근성 (Accessibility)**:
  - 시멘틱 태그인 `<section>` 또는 `<div>`를 사용하되, 리스트 형태의 데이터이므로 역할에 맞게 구성합니다.
- **타입 안전성**:
  - TypeScript 인터페이스(예: `PokemonGridProps`)를 정의하여 props의 타입을 명확히 합니다.
